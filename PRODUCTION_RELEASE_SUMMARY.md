## 🚀 部署指南

### 快速启动
```bash
# 克隆仓库
git clone https://github.com/luoxiao6645/ai-agent.git
cd ai-agent

# 安装依赖
pip install -r requirements.txt

# 运行测试
python run_test_suite.py

# 启动服务
python api_server.py
```

### Docker部署
```bash
# 构建镜像
docker build -t ai-agent:v1.1.0 .

# 运行容器
docker run -p 8000:8000 ai-agent:v1.1.0
```
