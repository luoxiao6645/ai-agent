## 🚀 部署方式

### 快速体验
```bash
# Docker一键部署
git clone https://github.com/luoxiao6645/ai-agent.git
cd ai-agent
docker-compose up -d
```

### 生产部署
```bash
# 增强版部署
python enhanced_app.py

# 或使用Docker生产配置
docker-compose --profile monitoring up -d
```
